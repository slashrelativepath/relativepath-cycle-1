# **Underdog Devs Devops Class I**

## Goal- create local Ubuntu VM running an NginX webserver from:
* Windows
* Osx
* Ubuntu

# Usage

- install git

- clone the repo

- install a hypervisor

- write a file or command the provisions an ubuntu vm

- connect to the vm

- install the webserver

- test the webserver
